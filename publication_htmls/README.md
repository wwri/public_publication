# Publication HTMLs

Interactive HTML visualizations for research publications organized by project.

## Projects

### [road_networks/](road_networks/)
Evacuation route analysis for U.S. Census-designated places.
- Bivariate maps showing egress capacity vs wildfire risk
- ~30,000 communities analyzed

### [fsim_nps_places/](fsim_nps_places/)
Wildfire risk and significance layers for cultural-resource places nationwide (interactive map aligned with FSIM/NRHP linkage analysis).
